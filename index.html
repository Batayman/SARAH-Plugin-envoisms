Ce module permet d'envoyer des SMS à des contacts pré-enregistrés.

Structures des phrases :

Pour éviter de rentrer en conflit avec d'autres plugins, je ne crée qu'une seule variante par personne:

  • Sarah envoie un texto à Jean
  • Sarah envoie un texto à Robert
  • Sarah envoie un texto à Juliette
  • Sarah envoie un texto à ...

Pré-requis:

Etape 1 - Création du compte

  • Allez à l'adresse https://www.pushbullet.com/ pour créer un compte.
  • Téléchargez l'application sur votre téléphone.
  • Allez à l'adresse https://www.pushbullet.com/account, cliquez sur "Create Access Token" et gardez la page ouverte nous en aurons besoin.

Etape 2 - Configuration du plugin (.prop)

  • Ouvrez l'interface de SARAH, cliquez sur configurer l'extension (du plugin sms) et collez l'access token qu'on à créé juste avant.
  • Allez dans le dossier "config" présent dans le dossier du plugin, éxécutez le fichier "recupData.bat" et collez à nouveau l'access token lorsque c'est demandé.
  • Dans le fichier qui vient d'être créé (devices.json), identifiez le "nickname" correspondant à votre téléphone (par exemple pour moi c'est Galaxy S4) puis récupérez la valeur du "iden" qui se trouve quelques valeurs avant.
    Et le coller dans "target_device_iden" dans l'interface SARAH.
  • Dans le fichier second fichier qui vient d'être créé (user.json), récupérez la valeur du "iden" qui se trouve quasiment au début. Et le coller dans "source_user_iden" dans l'interface SARAH.

Etape 3 - Ajouter des contacts

  • Dans le dossier du plugin, ouvrez le fichier "listeContacts.js" et rajoutez vos contacts sous cette forme :
    • {"name" : "prénom", "number" : "0666666666"},
      Ne pas oublier la virgule à la fin de chaque ligne. Ne pas supprimer la ligne {"name" : "debug", "number" : "debug"}
  • Dans le dossier du plugin, ouvrez le fichier "sms.xml" et rajoutez une ligne pour chacun de vos contacts sous cette forme :
    • < item >envoie un texto à robert < tag > out.action.destinataire="robert"; < /tag > < /item >
    • < item >envoie un texto à prénom < tag > out.action.destinataire="prénom"; < /tag > < /item >
